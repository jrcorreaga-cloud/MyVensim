\doxysubsection{Detailed Description}
Abstract base class representing a transfer of values between two Systems. 

The \doxylink{classFlow}{Flow} class connects a source \doxylink{classSystem}{System} to a target \doxylink{classSystem}{System}. Derived classes must implement the \doxylink{classFlow_a619be0b590c78202127bc6ac7fb04029}{execute()} method to define the specific behavior and transfer rate.
